# Esercizi Python

Raccolta personale di esercizi in Python svolti durante il corso, organizzati per
lezione e per percorsi tematici a difficoltà crescente.

## Struttura

- **`lezione_01` … `lezione_24`** — esercizi svolti lezione per lezione.
- **`ripasso/`** — 12 sezioni di ripasso a difficoltà crescente (basi, cicli,
  liste e stringhe, dizionari, funzioni, ricorsione, lambda/map/filter, eccezioni,
  classi, ereditarietà, file/JSON, progetto finale). Vedi
  [`ripasso/esercizi_ripasso.md`](ripasso/esercizi_ripasso.md) per le tracce.
- **`Esercizi_Potenziamento1/`** — esercizi aggiuntivi di approfondimento.
- **`Esercizi_Recupero/`** — esercizi di recupero e consolidamento.
- **`Esercizi_sicurezza/`** — esercizi con focus sulla sicurezza.
- **`esercizi_esame_python/`** — esercizi in stile prova d'esame.

## Utilizzo

Ogni file è eseguibile in modo indipendente:

```bash
python lezione_01/nome_esercizio.py
```
