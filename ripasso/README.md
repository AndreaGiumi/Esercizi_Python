# Ripasso Python

Raccolta di esercizi di ripasso Python, organizzati in 12 sezioni a difficoltà
crescente basate sugli argomenti visti a lezione.

Ogni file `.py` corrisponde a una sezione di [`esercizi_ripasso.md`](esercizi_ripasso.md),
dove sono riportate le consegne complete.

## Contenuti

| File | Argomento |
| --- | --- |
| `01_basi.py` | Variabili, input/output, condizioni |
| `02_cicli.py` | Cicli `while` e `for` |
| `03_liste_stringhe.py` | Liste e stringhe |
| `04_dizionari.py` | Dizionari |
| `05_funzioni.py` | Funzioni |
| `06_ricorsione.py` | Ricorsione |
| `07_lambda_map_filter.py` | Lambda, `map` e `filter` |
| `08_eccezioni.py` | Gestione delle eccezioni |
| `09_classi_basi.py` | Classi: basi |
| `10_classi_ereditarieta.py` | Classi: ereditarietà |
| `11_file_json.py` | File e JSON |
| `12_progetto_finale.py` | Progetto finale |

Legenda difficoltà: ⭐ facile · ⭐⭐ media · ⭐⭐⭐ difficile
